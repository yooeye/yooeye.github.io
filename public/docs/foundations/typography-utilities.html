Typography Utilities

These classes can be used to specify font sizes and weights outside of what would already be applied with .fe_f_typography (see the implementation section). Their style rules are applied with the highest level of specificity.

Usage

These utilities are presented here to allow finer control of text without the need to create custom CSS purely for altering text.

Larger font sizes intended to be used for headings should instead use the Heading component.

Design

Overuse of these classes can degrade application consistency or prove visually distracting to parts of the application requiring emphasis.
