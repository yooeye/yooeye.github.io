How to Read and Use the Color Swatches

The colors on this page are categorized by specific use cases meant to provide a consistent visual aesthetic to components and pages. To facilitate discussion between design and development, design specifications should always refer to the semantic name rather than the HEX or RGB values.

$color-interaction ← SASS map name

  • default
    #0275D4
    rgb(2, 117, 212)
    map-get($color-interaction, default)
  • ← Semantic name (SASS map key)
    ← HEX color value
    ← RGB(A) color value
    ← SASS map variable access

See the implementation section for how to use the SASS maps and keys when writing application SASS styles. Using the SASS color maps greatly reduces the need to recode user interfaces if there are color changes and is always the most preferable way of referencing color values in Forge. The HEX and RGB values are only provided for setting up design tools or in cases where the use of SASS is not possible.

Color Principles

The color palette was carefully considered with the following in mind:

Accessibility

athenahealth products must meet certain accessibility standards, including contrast ratios the Web Content Accessibility Guidelines (WCAG) 2.1 specifies for text and background color combinations. Forge colors are level AA compliant when used in proper combinations.

Safety

The color red is reserved to indicate patient safety issues; in terms of scale of importance patient lives are the highest and thus our color choices need to reflect that.

Meaning

A deliberate choice was made to use color sparingly. When color is used, it is intended to have specific, holistic meaning, so that regardless of which product, widget, screen, or workflow a user moves through, the meaning stays consistent.

Brand

Although Forge does not strictly adhere to the corporate color palette (primarily because of accessibility issues), that was the starting point.

Implementation

When coding with Forge styles in SASS, all colors on this page should be referenced through the use of their respective map using the SASS map-get function. Learn more about SASS maps.

.my-pill-box {
    border: 1px solid map-get($color-border, default);
}

Do not directly code the hex or rgb(a) value into your style rules. This will result in CSS styles that are difficult to maintain.
