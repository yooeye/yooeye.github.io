RadioGroup

RadioGroup allows users to select one item from several options represented by radio buttons.

Usage

RadioGroup is a set of radio buttons representing several mutually-exclusive options. RadioGroup is useful for enforcing structured data conventions by eliminating variability in user input.

Consider using a Select instead when more than five mutually-exclusive options needs to be presented.

Design

The RadioGroup component automatically changes its layout according to the following rules:

  • Radio buttons are horizontally aligned if there are just two options
  • Radio buttons are vertically aligned if there are more than two options

Once a user interacts with RadioGroup, a value always remains selected. Do not use RadioGroup if it is acceptable for that form input to have no explicit choice.

Implementation

Each RadioGroup is required to have a unique id within the context of a form. RadioGroup uses this identifier to further generate id attributes for its children radio inputs and the radio inputs' labels.

RadioGroup will auto-generate its own id when used with FormField.
